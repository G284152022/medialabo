「Webコンテンツ作成」の成果

本テーマの学習目的

HTML、CSS、JavaScriptによってWebページの構造や見た目、インタラクティブな処理を記述するなどの基本的な方法がわかるようになる。 また、構文リファレンスから構文の書式や使用方法の情報を入手できるようになる。

プロジェクト課題について

プロジェクト課題では世界の天気を表示させるプログラムを作りました。CSSでは色などの主張が激しくならないように注意し、見やすいページを意識して作りました。 また、JavaScriptを使用してプルダウンメニューから都市名を選択し、その都市の天気について知ることができるようにしました。 調べたい各都市のデータをAJAX通信を使って取得し、検索結果を見やすくしました。2回目に違う検索結果を表示させたいときに1回目の検索結果を削除し2回の検索結果だけを表示しました。

第3回 JavaScript の基本

第3回目の授業ではJavaScriptについて学びました。これはWebページ上で複雑な機能を実現する言語でページの自動更新や動きのあるページを実現できます。 プログラムを通じて実現可能な便利機能のことをAPIといい、DOM APIやGeolocation APIなどがあります。

第4回 DOM と JavaScript

第4回目の授業ではJavaScriptのDOMについて学びました。DOMを使ってWebの内容を読み取ったり、コンテンツに変更を加えることが可能になりました。変更を加えるには要素の検索が必要で、 HTMLで書いたclassなどを指定できます。他にもJavaScriptを使ってCSSプロパティを変更することもできることを学びました。エラー原因を見つけるために、コンソールを使ってプログラムを作ることが大切です。 そして、次の授業で習うイベント処理と組み合わせることでボタンを押した時の表示内容を変えることだできるようになるそうです。

第5回 イベント処理

第5回の授業ではJavaScriptのイベント処理について学びました。イベントとはWebページ上で起こった動作などのことで要素などをクリックすることなどを指します。これらの動作を行った時に 色が変化したり、文字を変化させるなどWeb上で動きをつけることができることを知りました。デバックの方法は2種類あり、イベントハンドラの登録を確認する方法とウェブ開発ツールのデバッガを使う方法があります。

またinput要素ではボタンと組み合わせることで入力フォームを作ることが作ることができます。input要素のtype属性には色々な種類があり、テキストボックスの他にカレンダーやカラーピッカーなどから選択することもできる。

工夫点，苦心点，強調点，感想

このレポートの工夫した点は、CSSで色を青系に統一させたことです。色を統一させたことによってページにまとまりができ、見やすくなったと思います。他にも、レポートの文章などは色やフォントは変えずにそのままにすることによって読みやすくしました。 また、プロジェクト課題の方も同じように色を統一させました。Webページはどんなに動きがあっても、読みにくく見づらければ良いページとは言えないと思います。そこを意識せた上で自分らしさを出せたと思います。
次に苦心点についてです。Webページを作るのは初めてなので苦心点はたくさんありました。一つ目は数当てゲームで正誤判定をさせることです。その頃はメソッドのことをあまり理解していなく、自分一人でプログラムを作成することが大変でした。 何回か挑戦するうちにできるようになり、とても良い経験になりました。二つ目はプロジェクト課題で、一から自分のページを作ったことです。自分は発想力や想像力が乏しく、自分で何もない状況から作ることは苦手です。 そのせいで、どんな動きをつけたら良いのかわからなくなり、手が動かなくなることが多々ありました。それでも無事完成できてよかったです。また、与えられたデータの中から表示させたいデータ取り出すことも大変でした。
強調点は工夫点と同じでWebページの全体的な色です。シンプルにまとめつつ、自分が好きな色でまとめていました。プロジェクト課題では、プルダウンメニューを使うことで簡単かつ素早く自分の調べたい国の天気がわかるようにしました。 あまり、目立った強調点はないですが自分なりに作成することができました。
今回の「Webコンテンツの作成」の講義を通して、普段自分たちが何気に使っているサイトなどは、とてつもない技術と努力によって作られているのだと感心しました。今回、HTMLとCSS,JavaScriptは基礎しか学んでいないのに毎授業ついていくのが精一杯なのに、 普段見ているサイトはもっと見やすかったり、動きなどがたくさんあって自分はまだまだだと実感しました。講義を受ける前には、文字の色や大きさなども一から自分で作成しなければいけないとは思いもしませんでした。しかし大学生活はまだ残ってるので、もっと自分で勉強したいです。講義を受けていくうちに、できることや学ぶことが増え、自分の成長を感じることができました。 この経験を他の講義のプログラミングなどにも活かしていきたいと思います。
