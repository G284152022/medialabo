第2回目の授業ではCSSについて学びました。CSSは要素の見た目を装飾することができ、色を変えたり、フォントを変えることができることがわかりました。 CSSのプロパティは、HTMLの要素や属性と同様にたくさん種類があり、参考資料を確認しながら取り組むのが良い。また次の課題に取り組みました。

第3回 JavaScript の基本

第3回目の授業ではJavaScriptについて学びました。これはWebページ上で複雑な機能を実現する言語でページの自動更新や動きのあるページを実現できます。 プログラムを通じて実現可能な便利機能のことをAPIといい、DOM APIやGeolocation APIなどがあります。

JavaとJavaScriptは名前が似ているが別の仕組みを持つ言語。その大きな違いはインタプリタ言語とコンパイル言語です。また次の課題に取り組みました。

  • 練習問題3-6

第4回 DOM と JavaScript

第4回目の授業ではJavaScriptのDOMについて学びました。DOMを使ってWebの内容を読み取ったり、コンテンツに変更を加えることが可能になりました。変更を加えるには要素の検索が必要で、 HTMLで書いたclassなどを指定できます。他にもJavaScriptを使ってCSSプロパティを変更することもできることを学びました。エラー原因を見つけるために、コンソールを使ってプログラムを作ることが大切です。 そして、次の授業で習うイベント処理と組み合わせることでボタンを押した時の表示内容を変えることだできるようになるそうです。
